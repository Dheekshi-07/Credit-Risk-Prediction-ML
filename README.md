Credit Risk Prediction


Project Description

This project predicts whether a loan applicant is high risk or low risk using machine learning.


Tools Used

* Python
* Pandas
* Scikit-learn


Dataset

The dataset contains applicant details such as age, income, loan amount, and credit score.


Steps

* Data preprocessing
* Model training
* Prediction
* Accuracy evaluation


Output

The model predicts credit risk and prints accuracy score.
